💬 聊天浮窗演示页

点击右下角的聊天按钮，体验智联AI客服的聊天浮窗功能。这个浮窗可以嵌入到任何电商网站中，为客户提供即时的AI客服支持。

🔗 嵌入方式

在你的网站 </body> 之前添加以下代码即可嵌入客服浮窗：

<!-- 智联AI客服浮窗 -->
<link rel="stylesheet" href="https://你的域名/css/style.css">
<div id="zhilian-widget"></div>
<script>
window.ZHILIAN_CONFIG = {
  serverUrl: "https://你的域名",
  botName: "小智",
  primaryColor: "#4F46E5"
};
</script>
<script src="https://你的域名/js/widget.js"></script>

  • 支持商品咨询、订单查询、售后处理
  • 自定义品牌颜色和欢迎语
  • 对话记录持久保存
  • 无缝对接客服工作台
